# C64 Pro Mono

`C64_Pro_Mono-STYLE.woff2` is the unmodified C64 Pro Mono webfont from
[C64 TrueType v1.2.1/Style](https://style64.org/release/c64-truetype-v1.2.1-style),
released by Style on April 15, 2019. It reproduces the Commodore 64 character
set for monospace display.

The upstream redistribution terms are preserved verbatim in `license.txt`.
